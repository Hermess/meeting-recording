# Meeting AI Kit Product Intro Video

16:9 Remotion product introduction video for Meeting AI Kit / 智能妙记.

## Preview

```bash
pnpm video:studio
```

## Render

```bash
pnpm video:render
```

The MP4 is written to:

```text
apps/product-video/out/meeting-ai-kit-intro.mp4
```

Subtitle files:

```text
apps/product-video/out/meeting-ai-kit-intro.zh-CN.srt
apps/product-video/out/meeting-ai-kit-intro.zh-CN.vtt
```

## Still Check

```bash
pnpm video:still
```

The preview frame is written to:

```text
apps/product-video/out/preview.png
```
